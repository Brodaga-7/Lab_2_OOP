Lab 2: Інкапсуляція та властивості

Опис програми
Програма для управління каталогом розумних годинників. Демонструє принципи інкапсуляції, використання властивостей та валідації даних.

Функціонал
1. Додавання годинників з перевіркою введених даних (ціна > 0, довжина назви тощо).
2. Перегляд таблиці годинників з автоматичним розрахунком статусу гарантії.
3. Пошук та фільтрація за різними параметрами.
4. Симуляція роботи: зарядка, дзвінки, встановлення програм (з перевіркою заряду).

Структура класів
  SmartWatch.cs: Основний клас. Містить приватні поля, публічні властивості.
  WatchOS.cs: Перелік (Enum) операційних систем.
  Program.cs: Консольне меню та взаємодія з користувачем.
